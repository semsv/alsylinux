1. Скопировать на USB Flash Drive папку alsylinux
2. Для MS Windows запустить файл с флешки alsylinux/boot/bootinst.bat
2. Для Linux выполнить команду ./bootinst.sh /точка монтирования USB FLASH Drive/alsylinux/boot/extlinux.exe
2. Для Linux, как альтернативный способ установки, 
   находясь в каталоге "alsylinux/boot",
   набрать команду: "./extlinux.exe -i $PWD"
3. Все готово!
